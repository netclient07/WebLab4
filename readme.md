# Приложение прогноза погоды

Чистый `HTML + CSS + JavaScript` без фреймворков.

## Что реализовано

- запрос геолокации при первом открытии;
- если геолокация отклонена, основной город выбирается вручную;
- прогноз на `сегодня + 2 дня` (минимум 3 дня);
- добавление дополнительных городов, переключение между ними, удаление;
- кнопка `Обновить` делает новые HTTP-запросы без перезагрузки страницы;
- состояния интерфейса: загрузка / ошибка / успех;
- подсказки городов при вводе + валидация выбора существующего города;
- сохранение локаций и активного города в `localStorage`.

## API

- Геокодирование: `https://geocoding-api.open-meteo.com/v1/search`
- Погода: `https://api.open-meteo.com/v1/forecast`

## Локальный запуск

Откройте `index.html` в браузере.
Важно: геолокация работает в защищенном контексте (`https`) или на `localhost`.
